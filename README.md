# Python Basics

A small collection of practice exercises and examples covering Python fundamentals: basics, conditionals, loops, functions, OOP, decorators, virtualenv usage, exceptions, generators, and a few mini-projects.
In this repo , I am following Chai Aur Python Series from <a href="https://www.youtube.com/playlist?list=PLu71SKxNbfoBsMugTFALhdLlZ5VOqCg2s">ChaiAurCode</a> Youtube Channel 

## Project structure
- **Root:** Overview scripts and notes: [practice.py](practice.py), [python_Notes.md](python_Notes.md), [README.md](README.md)
- **Basics:** [01_basics/](01_basics/) — simple hello and intro scripts
- **Conditionals:** [02_conditionals/](02_conditionals/) — conditional exercises and solutions
- **Loops:** [03-loops/](03-loops/) — loop problems and solutions
- **Functions:** [04_functions/](04_functions/) — exercises on functions
- **OOP:** [05_oops/](05_oops/) — classes, inheritance, MRO, examples
- **Decorators:** [06_decorator/](06_decorator/) — decorator problems
- **Projects:** [07_projects/](07_projects/) — small project scripts (youtube_manager)
- **Virtual env:** [08_virtual/](08_virtual/) — `requirements.txt` and virtual environment helper
- **Exceptions:** [09_exception/](09_exception/) — custom exception examples
- **Generators:** [generators/](generators/) — generator examples

## Requirements
- Python 3.8+ recommended
- Optional: create an isolated virtual environment
